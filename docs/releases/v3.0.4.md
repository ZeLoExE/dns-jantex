## DNS Jantex 3.0.4 — Stability & Security Update

### مهم‌ترین تغییرات

- اجرای رابط کاربری بدون دسترسی Administrator
- درخواست UAC فقط برای Apply، Reset و Flush از طریق Helper محدودشده
- Smart Connect واقعی با Queryهای DNS به‌جای ICMP Ping
- بررسی سه نمونه و انتخاب Median برای نتیجه‌ی پایدارتر
- اصلاح Auto Flush؛ فقط هنگام فعال‌بودن و دقیقاً یک بار اجرا می‌شود
- حفظ زبان، Provider انتخاب‌شده و وضعیت آپدیت بعد از Restart
- ذخیره‌ی تنظیمات در `%LOCALAPPDATA%\DNS Jantex`
- نوشتن اتمیک JSON و نگه‌داری نسخه‌ی فایل خراب برای بازیابی
- انتخاب Adapter براساس Default Route ویندوز
- اصلاح Providerهای تکراری و کلید ترجمه‌ی Mullvad
- خاموش‌شدن امن Workerها بدون `QThread.terminate()`

### Auto Update

این Release شامل دو Asset الزامی است:

- `DNSJantex-Setup.exe`
- `DNSJantex-checksums.json`

نسخه‌های جدید اندازه و SHA-256 فایل Installer را قبل از اجرا بررسی می‌کنند. در
صورت اضافه‌شدن Code Signing Certificate، بررسی Authenticode نیز به‌صورت
هم‌زمان اجباری خواهد شد.

### نکته‌ی نصب

کاربران نسخه‌های قدیمی می‌توانند از گزینه‌ی **Check for Updates** استفاده کنند.
Windows فقط هنگام اجرای Installer یا تغییر DNS پنجره‌ی UAC نمایش می‌دهد.
